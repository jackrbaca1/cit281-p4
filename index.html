RESTful API

A simple yet powerful API for managing questions and answers, built with Node.js and Express. This project demonstrates RESTful principles and best practices for API design.

Project Overview

This project is a RESTful API built with Node.js and Express. It provides endpoints to perform CRUD (Create, Read, Update, Delete) operations on questions and answers. The API follows RESTful conventions and returns JSON responses.

Key Features:

  • RESTful API design with proper HTTP methods and status codes
  • CRUD operations for questions and answers
  • Modular code structure with separate routes, controllers, and models
  • Error handling middleware for consistent error responses
  • Clean and intuitive API documentation

Getting Started

Prerequisites

Before you begin, ensure you have the following installed:

  • Node.js (v14 or later)
  • npm (Node Package Manager)

Installation

1. Clone the repository:

git clone https://github.com/jackrbaca1/cit281-p4.git
cd cit281-p4

2. Install dependencies:

npm install

3. Start the development server:

npm start

The API will be available at http://localhost:3000

Example Usage

Get All Questions

// Request
GET /api/questions

// Response
{ "status": "success", "results": 3, "data": { "questions": [ { "id": 1, "question": "What is Node.js?" }, { "id": 2, "question": "What is Express?" }, { "id": 3, "question": "What is REST?" } ] } }

Create a New Question

// Request
POST /api/questions
Content-Type: application/json { "question": "What is MongoDB?" }

// Response
{ "status": "success", "data": { "question": { "id": 4, "question": "What is MongoDB?" } } }
