CIT 281 Project 4: RESTful API

This is a simple RESTful API for managing questions and answers. The API is built with Node.js and provides endpoints for CRUD operations.

API Documentation

Get All Questions

GET /cit/question

Returns an array of all questions.

Get All Answers

GET /cit/answer

Returns an array of all answers.

Get All Questions and Answers

GET /cit/questionanswer

Returns an array of all question-answer pairs.

Get a Specific Question by ID

GET /cit/question/:id

Returns a specific question by its ID.

Get a Specific Answer by ID

GET /cit/answer/:id

Returns a specific answer by its ID.

Get a Specific Question-Answer Pair by ID

GET /cit/questionanswer/:id

Returns a specific question-answer pair by its ID.

Add a New Question-Answer Pair

POST /cit/questionanswer

Adds a new question-answer pair.

Example request body:

{
  "question": "What is Node.js?",
  "answer": "Node.js is a JavaScript runtime built on Chrome's V8 JavaScript engine."
}

Update a Question-Answer Pair

PUT /cit/questionanswer/:id

Updates an existing question-answer pair.

Example request body:

{
  "question": "What is Node.js?",
  "answer": "Node.js is an open-source, cross-platform, back-end JavaScript runtime environment that runs on the V8 engine and executes JavaScript code outside a web browser."
}

Delete a Question-Answer Pair

DELETE /cit/questionanswer/:id

Deletes a question-answer pair by its ID.

Running the Server Locally

To run this server locally:

# Install dependencies
npm install

# Start the server
node p4-server.js

The server will be available at http://localhost:3000
