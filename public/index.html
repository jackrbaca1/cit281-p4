RESTful API Documentation

Introduction

This is a RESTful API for managing questions and answers. The API provides endpoints to create, read, update, and delete questions and answers.

Base URL: /api

Questions

GET /api/questions

Get all questions

Example Request

GET /api/questions

Example Response

{
  "status": "success",
  "results": 3,
  "data": {
    "questions": [
      { "id": 1, "question": "What is Node.js?" },
      { "id": 2, "question": "What is Express?" },
      { "id": 3, "question": "What is REST?" }
    ]
  }
}
GET /api/questions/:id

Get a specific question by ID

Example Request

GET /api/questions/1

Example Response

{
  "status": "success",
  "data": {
    "question": {
      "id": 1,
      "question": "What is Node.js?"
    }
  }
}
POST /api/questions

Create a new question

Example Request

POST /api/questions
Content-Type: application/json

{
  "question": "What is MongoDB?"
}

Example Response

{
  "status": "success",
  "data": {
    "question": {
      "id": 4,
      "question": "What is MongoDB?"
    }
  }
}
PATCH /api/questions/:id

Update a question

Example Request

PATCH /api/questions/1
Content-Type: application/json

{
  "question": "What is Node.js runtime?"
}

Example Response

{
  "status": "success",
  "data": {
    "question": {
      "id": 1,
      "question": "What is Node.js runtime?"
    }
  }
}
DELETE /api/questions/:id

Delete a question

Example Request

DELETE /api/questions/1

Example Response

{
  "status": "success",
  "data": null
}

Response Status Codes

The API uses the following status codes:

Success

  • 200 OK - The request was successful
  • 201 Created - A new resource was created
  • 204 No Content - The request was successful but there is no content to return

Client Errors

  • 400 Bad Request - The request was invalid or cannot be served
  • 401 Unauthorized - Authentication is required
  • 403 Forbidden - The request is understood but it has been refused
  • 404 Not Found - The requested resource could not be found
  • 405 Method Not Allowed - The requested method is not allowed for this resource

Server Errors

  • 500 Internal Server Error - Something went wrong on the server
  • 502 Bad Gateway - The server received an invalid response from the upstream server
  • 503 Service Unavailable - The server is currently unavailable
